Projet flutter

Projet d'étude réalisé entre février et mars 2023

Projet réalisé en équipe : MICHEL Théo, VERSEL Romaric, LEROUX Mathis

Technologies utilisées : Flutter, dart, NodeJS


Nous avons développé une application mobile qui est un systeme de gestion de rendez-vous comportant une page de login, la création et la visualisation d'une visite selon le médecin connecté.


Taches confiés :

  • Création d'une API sous NodeJS
  • Création de la page de login
  • Création de la page de visualisation des visites

    • Fonctionnalités réalisées :

  • Connexion
  • Visualisation des visites selon le médecin connecté
  • Création de visite
